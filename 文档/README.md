# 文档目录说明

## 目录结构

### 00-当前生效资料
当前应优先阅读、持续维护的正式资料。

- `统一PRD资料包/` — 当前主用 PRD 体系资料包

### 90-历史废弃方案
历史方案、旧版 PRD、过程底稿归档区，仅供追溯，不再作为当前实施依据。

- `01-旧版PRD/`
- `02-历史底稿/`
- `03-历史规划/`

### 其他目录
- `分析报告/` — 分析类输出
- `开发指南/` — 开发与迁移操作指南
- `AGENTS.md` / `FEISHU_SETUP_GUIDE.md` — 配套说明

## 使用规则

1. 讨论当前产品方案、功能边界、研发拆解时，默认只看 `00-当前生效资料/统一PRD资料包/`。
2. 历史文件如果仍需保留，标题统一标记为 **[已废弃方案]**，避免被误当作现行方案。
3. 新增正式 PRD、功能规格、开发任务书、测试验收文档，统一放入 `00-当前生效资料/统一PRD资料包/` 对应层级。
4. 历史资料只归档，不在原目录继续混放。
