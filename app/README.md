# 高压线束成本核算工作台 (Automotive High-Voltage Harness Cost Accounting Workbench)

## 项目简介
本项目是一个专门用于汽车高压线束成本核算的动态模型应用。
- **定位**: 汽车高压线束成本核算动态模型
- **特性**: 离线优先的 PWA 应用，确保在各种环境下都能稳定运行
- **核心功能**: 支持逐线束精算、设变报价以及金属价格联动

## 技术栈
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite 6
- **UI 组件库**: Semi Design (字节跳动开源)
- **图表库**: ECharts
- **本地存储**: Dexie.js (基于 IndexedDB)
- **表格处理**: SheetJS (xlsx) 用于 Excel 导入与导出
- **状态管理**: Zustand
- **离线支持**: PWA (使用 Workbox)

## 功能概览
1. **项目管理**: 支持创建、查看项目列表，提供 4 步快速创建向导。
2. **仪表盘**: 包含 KPI 卡片、成本构成饼图、线束成本清单表。
3. **线束详情**: 展示成本瀑布图，细化材料、包装、运费明细。
4. **线束编辑**: BOM 编辑、工时/包装/运费调整，支持实时成本预览。
5. **报价工作台**: 集成吉利报价模板、设变报价、金属联动计算。
6. **Excel 导出**: 支持吉利模板、内部核算表、设变对比表及完整报价包。
7. **全局设置**: 配置默认费率、金属基准价等核心参数。

## 成本核算模型
应用遵循以下精确核算公式：
- **材料成本**: BOM 逐行计算
    - 导线: `铜重 × 铜价 + 铝重 × 铝价 + 非金属部分`
    - 其他零件: `单价 × 用量`
- **废品费**: `材料成本 × 1%`
- **人工费**: `工时 × 35 元/h`
- **制造费**: `工时 × 46.69 元/h`
- **管理费**: `(材料 + 人工 + 制造) × 6%` (基数不含废品)
- **利润**: `(材料 + 废品 + 人工 + 制造 + 管理) × 5.6627%`
- **出厂价**: `材料 + 废品 + 人工 + 制造 + 管理 + 利润`
- **到厂价**: `出厂价 + 包装费 + 运输费`
- **单车成本**: `Σ(到厂价 × 装车比)`
