## 📊 跨表格引用说明

### 核心引用关系

```
项目评估汇总（昆山90%）【主表】
    │
    ├── 引用 配置明细 → 材料成本、回路数
    ├── 引用 设备投资明细 → 设备成本
    ├── 引用 项目专用模具 → 模具成本
    ├── 引用 项目工装投入 → 工装成本
    ├── 引用 研发费用 → 研发成本
    └── 引用 包装物流费用 → 物流成本
```

### 常见跨表引用公式

| 公式 | 说明 |
|------|------|
| `=配置明细!P24` | 引用配置明细表P24单元格 |
| `=设备投资明细!K30` | 引用设备表K30单元格 |
| `=SUM(F5:K5)` | 汇总F5到K5的和 |
| `=F6/F4` | 回路单价 = 销售单价 / 回路数 |
