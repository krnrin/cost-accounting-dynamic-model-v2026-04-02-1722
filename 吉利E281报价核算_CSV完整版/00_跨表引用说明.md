## 关键跨表引用说明

### 1. 项目评估汇总（昆山90%）- 主表
- **位置**：汇总表，被其他多个工作表引用
- **关键数据行**：
  - 第5行：目标销售数量
  - 第15行：单套材料成本（BOM材料成本）
  - 第20-22行：设备成本（共用/专用）
  - 第23-32行：制造费用、研发费用、包装物流费用

### 2. 设备投资明细 → 项目评估汇总
- **引用内容**：装车比、总量纲、数量
- **用途**：计算单套设备成本
- **公式示例**：`=项目评估汇总（昆山90%）!P5`

### 3. 项目专用模具 → 项目评估汇总
- **引用内容**：装车比、总量纲
- **用途**：计算模具分摊成本

### 4. 项目工装投入 → 项目评估汇总
- **引用内容**：装车比、总量纲、数量
- **用途**：计算工装分摊成本

### 5. 研发费用 → 项目评估汇总
- **引用内容**：总量纲
- **用途**：计算单套研发成本

### 6. 包装物流费用 → 项目评估汇总
- **引用内容**：装车比
- **用途**：计算单套物流成本

### 7. 配置明细 → 项目评估汇总
- **引用内容**：回路数、装车比
- **用途**：计算工时和人力配置

## 使用建议（Notion AI）

### 1. 导入CSV到Notion
1. 在Notion中创建数据库
2. 选择 "Import" → "CSV"
3. 上传对应的数据CSV文件

### 2. 查看公式
- 公式保存在对应的 `_公式.csv` 文件中
- 公式以文本形式显示，可以搜索和查看

### 3. 理解跨表计算
- 参考本说明文档的"跨表格引用关系"部分
- 主表是【项目评估汇总（昆山90%）】
- 其他工作表的数据通常引用主表

### 4. 关键公式类型
| 公式类型 | 说明 | 示例 |
|---------|------|------|
| 单元格引用 | 引用同一工作表的单元格 | `=A1+B1` |
| 跨表引用 | 引用其他工作表的数据 | `=项目评估汇总（昆山90%）!P5` |
| 汇总计算 | 对某列/行求和 | `=SUM(A1:A10)` |
| 分摊计算 | 成本分摊到单套产品 | `=总成本/产量` |

## 注意事项

1. **CSV格式限制**：CSV不支持Excel的原生公式，公式已转换为文本形式
2. **多工作表**：原Excel有12个工作表，已分别导出为独立的CSV
3. **数据一致性**：所有数据CSV中的数值都是计算后的最终结果
4. **公式查看**：如需查看原始公式，请参考对应的 `_公式.csv` 文件

## 数据示例

### 单套成本计算公式（项目评估汇总表）
```
单套设备成本 = 设备总投资 / 总产量
单套模具成本 = 模具总投资 / 总产量  
单套工装成本 = 工装总投资 / 总产量
单套研发成本 = 研发总费用 / 总产量
```

### 材料成本计算（配置明细表）
```
材料成本 = 导线成本 + 非导线成本
导线成本 = 铜重 × 铜价 + 铝重 × 铝价
```
