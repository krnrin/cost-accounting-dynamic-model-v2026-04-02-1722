## 关键计算公式说明

### 1. 单套设备成本计算

```
单套共用设备成本 = 共用设备总投资 / 总产量
                = 19390669.35 / 600000
                = 19.39 元/PCS

单套专用设备成本 = 专用设备总投资 / 总产量
                = 1639500 / 600000
                = 2.73 元/PCS
```

### 2. 单套模具成本计算

```
单套模具成本 = 模具总投资 / 总产量
            = 1796000 / 600000
            = 2.99 元/PCS
```

### 3. 单套工装成本计算

```
单套工装成本 = 工装总投资 / 总产量
            = 1317800 / 600000
            = 2.20 元/PCS
```

### 4. 单套研发成本计算

```
单套研发成本 = 研发总费用 / 总产量
            = 582000 / 600000
            = 0.97 元/PCS
```

### 5. 材料成本计算（配置明细表）

```
材料成本 = 非导线成本 + 导线成本

导线成本 = 铜重 × 铜价 + 铝重 × 铝价
        = 铜重 × 69.43 + 铝重 × 铝价
```
